# Adamant: a webstore powered by React & GraphQL

Adamant is a side project utilizing technologies including:

- React
- Styled Components
- NextJS
- Apollo
- GraphQL
- NodeJS
- Stripe API

## Testing

Jest and Enzyme are used for testing.
